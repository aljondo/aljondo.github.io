Shockwa.ve

2015

While taking Northeastern's Human Computer Interaction Course, my team and I built a protoype (front-end only) of a web application called Shockwa.ve which would allow users online to listen to music together as if they were in the same room.

Developing with Javascript leveraging the Bootstrap CSS library.
